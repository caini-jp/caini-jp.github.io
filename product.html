①. [3Dprint] 3Dprint Bike
       [3Dプリント] 3Dプリント自転車
       [3D打印] 3D打印自行车


1. Overview / 概要
3D-printed bicycle model, built with Rhino and sliced using cura. The model is a Stereo Lithography appearance type, and it was spray-painted after printing.
3Dプリントの自転車モデル。Rhinoでモデリングし、curaでスライス処理を行い、タイプはステレオリソグラフィー外観モデルです。プリントした後に塗装して仕上げました。
3D 打印自行车，使用 Rhino 建模，cura 切片处理，类型为立体光刻外观模型。打印完成后进行了喷漆上色。
2. About
Creator: CAI Ni, LIU Jing, CHEN Mingxing

       [3Dプリント] 法務サービス端末機
       [3D打印] 法律服务终端机


<1. Overview / 概要

3D 打印终端机，使用 Rhino 建模、cura 进行切片。类型为熔融沉积成型。打印完成后进行了喷漆上色。

④. [Product Design] Legal services platform
       [プロダクトデザイン] 法務サービス端末機
       [产品设计] 法律服务终端机




1. Overview / 概要
Undergraduate graduation project at Wuhan Institute of Technology. The project aims to provide convenient legal services through the internet. Designed the working process and created the renderings using Rhino, KeyShot and Photoshop, and printed it with a 3D printer.
本プロジェクトは、インターネットを通じて便利な法律サービスを提供することを目的としています。Rhino、KeyShot、Photoshop を用いて作成し、3Dプリンターでモデルを出力しました。
在武汉工程大学完成的本科毕业设计项目。该项目旨在通过互联网提供便捷的法律服务。我负责设计工作流程，并使用 Rhino、KeyShot 和 Photoshop 制作效果图，最终通过 3D 打印机输出实体模型。
2. About
Author: CAI Ni

⑦. [Product Design] Table set
       [プロダクトデザイン] テーブルセット
       [产品设计] 桌椅套装



1. Overview / 概要
This is a table-and-chair set designed for children. Soft materials are used for the table edges, table legs, and chair legs to reduce the risk of injury from accidental bumps. Additionally, the chair’s backrest can also function as a tabletop, enhancing both usability and versatility.
本作品は子ども向けにデザインしたテーブルとチェアのセットです。テーブルの縁や脚、チェアの脚には柔らかい素材を使用し、衝突によるケガを防ぐ工夫をしています。さらに、チェアの背もたれはテーブル面としても利用でき、機能性を高めています。
这是一套为儿童设计的桌椅套装。桌子的边缘和桌脚、椅子的脚部均采用柔软材质，以减少碰撞造成的伤害。此外，椅子的靠背也可作为桌面使用，提升了整体的功能性与趣味性。
2. About
Creator: CAI Ni, GUO Jingyu

这是一款为儿童设计的订书机和剪刀，其外观造型借鉴了鲸鱼和鲨鱼的形象，让文具更加有趣、安全且富有亲和力。
